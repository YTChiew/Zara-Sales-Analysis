# Zara-Sales-Analysis

## Project Overview
This project is to analyze Zara sales dataset to identify trends, sales performance, and insights for decision making and marketing strategy.

## Project Overview
This project analyzes Zara sales data to uncover insights about:
- Top-selling products
- Monthly revenue trends
- Sales performance by store location
